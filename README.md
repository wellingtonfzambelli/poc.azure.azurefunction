# About
It's a project to show how to use a Azure Function (TimerTrigger, HttpTrigger, MySqlChangesTrigger, MySqlSelect)

# Stacks of this project
- .NET 8
- Azure Function
- MySQL
- Postman _(for API testing)_
- Visual Studio 2022
